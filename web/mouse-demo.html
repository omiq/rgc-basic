RGC-BASIC — Mouse demo

New mouse API in the canvas WASM build: GETMOUSEX(), GETMOUSEY(), ISMOUSEBUTTONPRESSED/DOWN/RELEASED/UP(n), MOUSESET x, y, SETMOUSECURSOR n, HIDECURSOR, SHOWCURSOR. This page auto-runs examples/gfx_mouse_demo.bas.

Left button
paint pixels with PSET GETMOUSEX(), GETMOUSEY() while held
Right button
clear the drawing area
Middle button
toggle cursor visibility (HIDECURSOR / SHOWCURSOR)
Left + Right
warp BASIC's reported pointer to centre (MOUSESET 160, 100) — watch the X/Y status row jump to 160,100; the OS cursor itself can't be moved by a webpage
Vertical zone
cursor shape auto-changes via SETMOUSECURSOR (pointer / crosshair / default)
