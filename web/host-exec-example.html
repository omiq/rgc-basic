WASM EXEC$ / SYSTEM — host hook demo

This page defines Module.rgcHostExec before loading basic.js. The BASIC program calls EXEC$ / SYSTEM with strings like demo:echo …; only JavaScript you write runs (see docs/wasm-host-exec.md). Serve this file from the web/ directory (same origin as basic.js / basic.wasm), e.g. python3 -m http.server 8765 in web/, then open http://127.0.0.1:8765/host-exec-example.html. Build first: make basic-wasm.
