RGC-BASIC — Canvas (40×25 / 80×25 PETSCII)

Retro Game Coders BASIC

Click the canvas for GET / INKEY$ and for PEEK(56320+…) keyboard (WASD). Mouse: GETMOUSEX/GETMOUSEY, IsMouseButtonDown(0) (0=left, 1=right, 2=middle), MOUSESET x,y, SETMOUSECURSOR n, HIDECURSOR/SHOWCURSOR — the page polls pointer position over the framebuffer each animation frame. Use #OPTION … at the start of a line, or 10 #OPTION … with a line number (numberless programs accept both). Upload to VFS / Download from VFS move files into or out of WASM memory for OPEN / PRINT# (e.g. upload gfx_jiffy_game_demo.bas from the repo examples/, then Run with /gfx_jiffy_game_demo.bas in the textarea or load via VFS). #OPTION columns 80 for wide mode. Rebuild: make basic-wasm-canvas. Open this page as http://127.0.0.1:PORT/canvas.html or http://localhost:PORT/… — some browsers treat http://0.0.0.0:PORT/ oddly for WASM. After deploy, use canvas.html?cb=anything once if you see ASM_CONSTS errors (forces JS+WASM to load as a pair). Diagnostics: canvas.html?debug=1 — console shows build stamp and stacks; Module.canvasDebugDumpStacks() anytime.
